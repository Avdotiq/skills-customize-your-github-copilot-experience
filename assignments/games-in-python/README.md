# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Implement a fully functional Hangman game that allows players to guess letters and try to reveal a hidden word before exhausting their attempts. The game should provide clear feedback on progress and determine when the player wins or loses.

#### Requirements

Your game must:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in the format: `_ _ _` (underscores for unguessed letters)
- Track incorrect guesses remaining and display them to the player
- End the game when the word is guessed or when attempts are exhausted
- Display appropriate win/lose messages based on the game outcome
